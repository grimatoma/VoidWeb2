Proposed UI from Locked Spec

15 spec-faithful mockups + 5 creative explorations.

HTML/CSS mockups built directly from the locked prompts in CLAUDE_DESIGN_PROMPTS.md, captured as PNG via headless Chrome at the spec viewports (1440×900 desktop, 390×844 mobile). Source HTML lives in concepts/proposed/src/ — these double as a head start on the actual game shell, not just disposable renders.

T0 vertical slice (8)

Map default, Body Sheet, Ops, Production, Fleet, Build Drawer, Buy Ship, Trade — the spec calls these out as the recommended T0 handoff bundle.

01 Map (desktop)

Anchor screen. Orbital canvas, mini alerts, fleet strip, Selected inspector. Sets the visual language.

02 Body Detail Sheet

Universal selection result. Bottom sheet at half state. Survey readings, warehouse, buildings.

03 Ops

90-second check-in. Critical alerts hero, AFK summary, idle ships, hot list, sticky resolve CTA.

04 Production

Chain-design + build-placement. Visual flow with bottleneck-tinted edges, building list right.

05 Fleet

Ship management table. Hangar summary KPIs, status filters, inline reassign/recall actions.

06 Build Drawer

Slide-up recipe browser. T0 unlocked, T1+ shown locked with unlock-tier label.

07 Buy Ship

Earth shipyard modal. T2 view with all 6 hulls. Slot icons, stat tiles, ≥25% credits confirm flag.

08 Trade — Earth Market

Resource × buy/sell × owned × 24h net table with quantity steppers. Account KPIs at top.

T1+ & cross-cutting (4)

Colonies (Lunar Foothold), AFK Return modal, Tier-Up modal, Survey Region Picker — fire after the T0 vertical slice ships.

09 Colonies

Lunar Foothold T1+. Pop tier ladder, life-support bars, Settled bundle progress, building list.

10 AFK Return Modal

Hero $-delta + raw counts. What happened / what stopped. Resolve top issue CTA. No "welcome back" warmth.

11 Tier-Up Modal

Plain modal at v1 (cinematic deferred to Stage 2). T1 unlocks: 5 recipes, 1 ship, 2 prefab kits.

12 Survey Region Picker

Setup-only survey. Pick region + probe focus, then probe time elapses idle. No active minigame.

Mobile + light spec (3)

Map mobile (390px), Milestones (light spec), Research (light spec).

13 Map (mobile)

390×844. Mode selector, orbital canvas, alert peek, fleet peek, bottom 5-tab nav.

14 Milestones

T0–T7 ladder, T1 gate progress (124/200 metals + H₂ ≥ 50), completed milestones.

15 Research

Tech tree. Logistics branch detailed. Empty active-research-queue state in side rail.

Creative explorations · NOT part of locked spec (5)

Probes into areas the locked spec defers — Stage-2 cinematic, popover scope, late-game scale, long-arc surfaces, alternate Map modes. New prompts appended to CLAUDE_DESIGN_PROMPTS.md.

16 Tier-Up Cinematic

Stage-2 polish target. Letterbox frame, lunar habitat scene, single poetic line. Counter-proposal to the v1 plain modal.

17 Sources & Sinks Popover

Hover popover on a resource. Producers, consumers, net rate, time-to-cap, "what changes if stall resolved". Resolves a P0 open question.

18 Multi-System Overview

T5+ late-game scale test. 7 colonies across 4 sectors, 84 routes, system digest, recent feed.

19 Weekly Recap

Long-arc check-in (7 days). Credits trend, throughput vs last week, what fixed / what stalled, suggested focus.

20 Map · Heat Mode

The Heat map mode the spec mentions but doesn't define. Body tints by net throughput, route arcs by saturation, suggested rebalances.

Grouped View Matrix

Six Real Views, Three Styles Each

This set keeps the same UI contents and actions across styles, so the comparison is about product direction instead of random cool screenshots. The view definitions live in UI_VIEWS.md.

System Command

Main strategic overview: orbital map, route windows, alerts, selected-object inspector, and active fleet strip.

Light mission-control shell around a dark orbital command map.

System Command / Dark

Mobile Operations

Readable vertical management with light cards and dark map module.
