# 04 — Staged Delivery Plan

## Stage 0 — Direction Validation

Goal: confirm visual and interaction direction cheaply before architecture lock-in.

Deliverables:
- concept gallery comparison,
- live orbital visual sketch,
- UI page inventory,
- desktop/mobile layout principles.

Decision gate:
- commit baseline visual direction,
- confirm map readability,
- confirm mobile is task-first (not compressed desktop).

## Stage 1 — North Star Vertical Slice

Goal: prove the core fantasy in one coherent playable path.

Scope:
1. Command page with live-feeling map and actionable alerts.
2. Fleet route setup for one ship and one recurring delivery.
3. Industry chain: ore -> metals.
4. Colony oxygen warning with one-click import fix.

Success criteria:
- New player finds top bottleneck in < 20 seconds.
- Critical warning resolved in <= 2 interactions.
- Route setup clearly communicates ETA/fuel tradeoff.

## Stage 2 — System Depth

Scope:
- richer survey data layers,
- route templates and repeat logic,
- expanded colony tier requirements,
- initial automation rules,
- broader industry dependencies.

## Stage 3 — Scale & Mastery

Scope:
- multi-colony balancing,
- local shipbuilding dependencies,
- robust automation exception handling,
- long-horizon strategic objectives.

## Stage-by-Stage Principle

Each stage must answer one major question before adding complexity.
