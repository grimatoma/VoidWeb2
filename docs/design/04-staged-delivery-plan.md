# 04 — Staged Delivery Plan

## Stage 0 (Direction Validation)

Goal: verify aesthetic and interaction direction cheaply.

Deliverables:
- concept gallery comparisons,
- one live orbital preview,
- page and action inventory.

## Stage 1 (North Star Vertical Slice)

Goal: prove the core fantasy is compelling.

Scope:
1. Command page with live-feeling map + actionable alerts.
2. Fleet route assignment for one ship.
3. Industry chain from ore -> metals.
4. Colony oxygen shortage warning with one-click import fix.

Success criteria:
- New player can identify the top bottleneck in < 20 seconds.
- Critical warning can be resolved in <= 2 interactions.
- Route decision clearly shows ETA vs fuel tradeoff.

## Stage 2 (System Depth)

Scope:
- richer survey outcomes,
- route templates and repeat logic,
- expanded colony tier requirements,
- first automation rules.

## Stage 3 (Scale & Mastery)

Scope:
- multi-colony balancing,
- advanced goods and shipyard dependencies,
- broad automation and exception handling,
- long-horizon objectives.
